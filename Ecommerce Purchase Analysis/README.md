# Ecommerce Purchase Analysis using Pandas

## Project Overview
This project focuses on beginner-friendly data analysis using Python Pandas on an Ecommerce Purchase dataset.

## Tools Used
- Python
- Pandas
- Google Colab

## Questions Solved

1. Display Top 10 Rows of The Dataset
2. Check Last 10 Rows of The Dataset
3. Check Datatype of Each Column
4. Check null values in the dataset
5. How many rows and columns are there in our Dataset?
6. Highest and Lowest Purchase Prices
7. Average Purchase Price
8. How many people have French 'fr' as their Language?
9. Job Title Contains Engineer
10. Find The Email of the person with a specific IP Address
11. Mastercard users with purchase above 50
12. Find email using Credit Card Number
13. AM vs PM Purchases
14. Credit cards expiring in 2020
15. Top 5 most popular email providers

## Pandas Concepts Practiced

- read_csv()
- head()
- tail()
- info()
- isnull()
- shape
- filtering
- string methods
- value_counts()
- mean()
- max()
- min()

## Goal
The goal of this project is to improve practical Pandas and data analysis skills using real-world datasets.
